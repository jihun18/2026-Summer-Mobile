## 2026-06-30 학습 요약

* **다형성(Polymorphism)과 타입 캐스팅:** 부모 클래스 참조 변수로 자식 객체들을 리스트(`List<Shape>`)로 묶어 일괄 제어하는 다형성을 실습하고, `is` 키워드를 활용한 타입 검사 및 `as`를 이용한 명시적 다운캐스팅(Downcasting) 메커니즘을 정립함.
* **추상 클래스(Abstract Class)와 오버로딩:** 인스턴스화가 불가능한 `abstract class`의 개념을 이해하고, 자식 클래스에서 반드시 구현해야 하는 추상 메서드 오버라이딩과 매개변수의 타입·개수에 따라 함수를 다르게 정의하는 메서드 오버로딩(Overloading)을 구현함.
* **인터페이스(Interface)와 디폴트 메서드:** 코틀린 인터페이스 내에서 본문이 있는 디폴트 메서드(`land()`)를 구현하는 방법과 구현부가 없는 추상 메서드를 자식 클래스(`Duck`)에서 강제 오버라이딩하여 다중 구현의 기반을 학습함.
* **클래스 위임(Class Delegation):** 코틀린의 독창적인 `by` 키워드를 활용하여 인터페이스의 구현 책임을 직접 작성하지 않고, 이미 해당 인터페이스를 구현한 다른 클래스 객체에게 통째로 넘겨주는 상속 대안 플러그인 구조(`Printer by printer`)를 마스터함.
* **람다 함수와 고차 함수(High-order Function):** 익명 함수 표현식인 람다에서 단일 매개변수를 `it`으로 축약하는 문법을 익히고, 함수가 또 다른 람다 함수를 반환하거나 매개변수로 람다를 전달받는 고차 함수(`multiplier`, `testFun`)의 고급 제어 흐름을 체득함.
