1. 프로젝트 배경 및 선정 동기

  • 자취 생태계 진입에 따른 현실적 니즈 해결
    – 독립 후 직면할 "오늘 뭐 먹지?", "초보 요리 가이드"에 대한 개인적·실질적 고민에서 출발
  • 실제 사용자 경험(UX) 기반의 취약점 분석
    – 조리 과정 중 다수 실사용하며 체감한 직관적 단계별 가이드라인의 가치 검증
  • 전공 역량 증명을 위한 기술적 레퍼런스
    – 대규모 조회 중심 데이터 레이아웃에 알고리즘 및 로컬 DB 인터랙션을 더하기 적합한 주제

4. 핵심 기술적 챌린지 및 구현 포인트

① 실시간 검색 제어

• TextWatcher 부착, 자음/모음 단위 감지
• 전방 매칭 알고리즘 기반 실시간 연관 키워드 추천 갱신
• 포커스 진입 시 키패드 강제 활성화 제어

② 냉장고 털기 알고리즘

• ChipGroup 컴포넌트 동적 생성/삭제 인터랙션
• 코틀린 intersect 컬렉션 함수 활용
• 보유 재료 데이터 교집합 매칭 정렬 로직 설계

③ 장보기 로컬 DB CRUD

• Jetpack Room DB 라이브러리 탑재
• 체크리스트 영속성 저장 및 완전한 CRUD 구현
• 리스트 데이터 0개 도달 시 Empty State UI 예외 대응
