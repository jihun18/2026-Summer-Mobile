## 📅 2026-06-29 학습 요약

* **배열(Array)의 선언과 활용:** 람다식을 이용한 배열 초기화 방식(`Array(크기) { 초기화_로직 }`)을 실습하고, 기본 타입 배열인 `IntArray`와 제네릭 배열인 `Array<Int>`가 서로 다른 객체 타입 클래스임을 비교 분석함.
* **배열 순회와 반복문 제어:** 기본 `for`문 활용 시 루프 내부에서 아이템 값 자체를 수정할 수 없는 특성을 파악하고, `indices`를 통한 인덱스 접근 방식과 `withIndex()` 구조 분해 선언을 통해 인덱스와 실제 데이터를 동시에 제어하는 방법을 마스터함.
* **표준 컬렉션 프레임워크(Collection):** 순서가 있고 중복을 허용하는 `List`, 순서가 없고 중복을 불허하는 `Set`, 그리고 `Pair` 객체나 `to` 키워드를 활용해 키-값 쌍을 저장하는 `Map`의 구조적 차이와 컬렉션별 루프 순회 기법을 습득함.
* **객체지향 프로그래밍(OOP) 및 생성자 체인:** 클래스 주 생성자 파라미터에 `val/var`를 선언하여 멤버 변수 선언을 축소하는 최적화 구조를 이해하고, `init` 블록과 주 생성자, 보조 생성자(`constructor`) 간의 위임 호출 체인(`:this()`) 흐름을 정립함.
* **고급 클래스 구조 (상속, Data, Companion):** `open` 키워드를 활용한 클래스/메서드 상속 및 `override` 구현, 기본 생성자 속성 기준 자동으로 동등성을 비교하는 `data class`, 그리고 클래스명으로 정적 멤버처럼 접근이 가능한 `companion object` 문법을 학습함.
